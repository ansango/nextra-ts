Conseguiremos algo como esto:

```powershell
echo "# Hello, world!" > some.md
tinymd some.md
Compile Markdown...
Done!
cat some.html
<h1>Hello, world!</h1>
```

## Instalar Rust

Antes de comenzar tenemos que instalar Rust, para ello ejecutaremos en nuestra terminal lo siguiente:

```powershell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Puedes revisar la documentación [aquí](https://rustup.rs/)

Para comprobar que se ha instalado correctamente, deberemos reiniciar la terminal y ejecutar lo siguiente:

```powershell copy
rustc --version
// -> rustc 1.64.0 (a55dd71d5 2022-09-19)
```
