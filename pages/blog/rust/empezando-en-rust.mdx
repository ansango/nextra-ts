## Instalar Rust

Antes de comenzar tenemos que instalar Rust, para ello ejecutaremos en nuestra terminal lo siguiente:

```powershell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Puedes revisar la documentación [aquí](https://rustup.rs/)

Para comprobar que se ha instalado correctamente, deberemos reiniciar la terminal y ejecutar lo siguiente:

```powershell
rustc --version
-> rustc 1.64.0 (a55dd71d5 2022-09-19)
```

## Objetivos

1. **Crear un nuevo proyecto en Rust**
   - Crear un nuevo proyecto Rust en la línea de comandos
   - Compilar y construir un proyecto simple de Rust "Hello, World"
2. **Variables y funciones**
   - Crear una función
   - Crear una variable de tipo entero
   - Imprimir una variable de tipo entero en la línea de comandos
3. **Strings y memoria**
   - Crear una variable de tipo string
   - Devolver una variable de tipo string desde una función
   - Concatenar dos cadenas
   - Imprimir un string en la línea de comandos
4. **La lógica del compilador**
   - Describir cómo funciona un compilador en general
   - Crear un vector
   - Leer y analizar los argumentos de la línea de comandos
   - Implementar un bloque de coincidencia
   - Pasar un argumento a una función
5. **Leer y escribir archivos**
   - Abrir un archivo
   - Leer un archivo línea por línea
   - Describir cómo funciona un compilador de Markdown
   - Escribir en un archivo
6. **Release del proyecto**
   - Construir una versión de un proyecto en Rust
